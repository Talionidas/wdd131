Site Plan

Site Name

Dead Cells Biomes Guide

As a Guide for the Videogame Dead Cells, which mostly focuses on the biomes of Dead Cells, a simple and straightforward Site name is important, since it already conveys the purpose of the Site.

Site Purpose

A guide through the various Biomes of Dead Cells, providing different pathing, the pros and cons of each path, as well as tips and tricks. This Guide is mainly for newer players to facilitate choice of Biome selection.

Scenarios

  1. Guide for difficulty
    • How does this guide make my runs easier?
    • Should I use the same pathing for every run?
    • Is there a path that is better given a certain playstyle (Brutality/Tactics/Survival)
  2. Guide for DLCs
    • What DLC path is the most interesting?
    • Is there a path to avoid at first?
    • Are DLCs friendly to all playstyles?
